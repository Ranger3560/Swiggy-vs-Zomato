# Swiggy-vs-Zomato

🍽️ Restaurant Delivery Platform Analytics & Prediction
A complete data analytics and machine learning project based on restaurant performance data from food delivery platforms like Swiggy and Zomato. This project performs exploratory data analysis (EDA), visualizations, business intelligence reporting, and predictive modeling to uncover insights about restaurant operations, customer engagement, ratings, delivery efficiency, and revenue generation.

📌 Project Overview
This project analyzes restaurant datasets containing:


Restaurant details


Cuisine information


Ratings from Swiggy & Zomato


Delivery performance


Revenue and profit estimates


Customer engagement metrics


Platform comparison insights


Amenities and operational features


The goal is to:


Understand restaurant performance trends


Compare Swiggy vs Zomato metrics


Identify factors affecting revenue and ratings


Build predictive models for restaurant revenue


Generate business insights through visualizations

⚙️ Installation
Clone the repository:
git clone https://github.com/your-username/restaurant-analytics.git
Move into the project directory:
cd restaurant-analytics
Install dependencies:
pip install -r requirements.txt

▶️ Usage
Run the Jupyter Notebook:
jupyter notebook
OR run the Streamlit dashboard:
streamlit run app.py
